📝 Registration Form UI

A clean and structured **registration form** built using HTML and CSS. This project focuses on organizing form elements and applying basic styling.

---

🚀 Live Demo

👉 https://harshanrepo.github.io/user-registration-form/

---

📌 Features

* Multiple input fields (text, email, password, number)
* Radio buttons and checkbox inputs
* File upload input
* Dropdown selection menu
* Clean and centered layout

---

🛠️ Built With

* HTML5
* CSS3

---

📚 What I Learned

* Structuring forms using fieldset and labels
* Handling different input types
* Styling form layouts using CSS

---

🔧 How to Run Locally

1. Clone the repository
2. Open `index.html` in your browser
